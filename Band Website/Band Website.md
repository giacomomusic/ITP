## Problems Faced / Decisions Made 

For this project everything in terms of the actual coding of the html file went perfectly. We were all able to combine our contributions to the file, use the other groups website as a base, and when viewed locally it all looked great. However when first posted to GitHub it did not work at all. We double checked everything and couldn't figure it out until Nick remembered the html file had to be named index.html. Once we tried that... we once again were given an error 404 as we made the mistake of naming it Index.html not index.html and the capital letter was the last problem in our way. once that change was made the website worked perfectly.
